Vandermode Matrix

Description

The function returns the Vandermode matrix of a numeric vector, x, whose columns are the vector raised to the powers 0:n.

Usage

vandermode(x, n)

Arguments

x

a numeric vector

n

a numeric scalar

Value

a matrix of size length(x) x n

Examples

vandermode(1:5, 4)
